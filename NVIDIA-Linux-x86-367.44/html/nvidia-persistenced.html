Chapter 27. Using the nvidia-persistenced Utility

A Linux daemon utility, nvidia-persistenced, is included with the NVIDIA Linux GPU driver. After installing the driver, this utility may be installed to run on system startup or manually with the command:

    % nvidia-persistenced

in a terminal window.

The utility is not installed to run on system startup by the installer. Sample init scripts for some common init systems are provided in /usr/share/doc/NVIDIA_GLX-1.0/sample/nvidia-persistenced-init.tar.bz2 to aid in installation of the utility.

Detailed help information is available via the --help command line option and via the nvidia-persistenced man page.

The source code to nvidia-persistenced is released under the MIT license and is available here: ftp://download.nvidia.com/XFree86/nvidia-persistenced/
